# ChallengMe! 📱

Aplicación Android de **ChallengMe!** — retos diarios personalizados 
con inteligencia artificial.

## ¿Qué es ChallengMe!?

App de ocio que propone un reto personalizado cada día según tus 
intereses y nivel de dificultad. Completa retos, acumula puntos 
y construye rachas de días consecutivos.

## Stack tecnológico

- **Lenguaje:** Kotlin
- **UI:** Jetpack Compose
- **Arquitectura:** MVVM + Clean Architecture
- **Backend:** ASP.NET Core (C#) + Azure
- **IA:** Google Gemini

## Funcionalidades principales

- Reto diario generado por IA según tus intereses
- Sistema de puntos y rachas
- Ranking global de usuarios
- Subida de foto como evidencia del reto completado
